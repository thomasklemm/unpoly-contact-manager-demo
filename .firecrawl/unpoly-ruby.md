# [Install](https://unpoly.com/install)  Install with Ruby

## Ruby on Rails

The [`unpoly-rails`](https://github.com/unpoly/unpoly-rails) gem implements the [optional server protocol](https://unpoly.com/up.protocol) for Ruby on Rails applications.


If you're using the **Asset Pipeline**, `unpoly-rails` also provides Unpoly's frontend files.


If you're using **Webpacker**, install the [`unpoly` npm package](https://unpoly.com/install/npm) to get Unpoly's frontend files.


## Roda

The [`roda-unpoly`](https://github.com/adam12/roda-unpoly) gem implements the [optional server protocol](https://unpoly.com/up.protocol) for Roda applications.


If your Roda app uses an asset bundler like Webpack, install the [`unpoly` npm package](https://unpoly.com/install/npm) to get Unpoly's frontend files.


If you don't want to use npm you can [load Unpoly from CDN](https://unpoly.com/install/cdn) or [download Unpoly directly](https://unpoly.com/install/download).


## Rack

The [`rack-unpoly`](https://github.com/adam12/rack-unpoly) gem implements the [optional server protocol](https://unpoly.com/up.protocol) for Rack applications.


If your Rack app uses an asset bundler like Webpack, install the [`unpoly` npm package](https://unpoly.com/install/npm) to get Unpoly's frontend files.


If you don't want to use npm you can [load Unpoly from CDN](https://unpoly.com/install/cdn) or [download Unpoly directly](https://unpoly.com/install/download).
